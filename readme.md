# Tem Quadra?

O sistema definitivo para agendamento de quadras esportivas. Conectando atletas a arenas de forma rápida e responsiva.

# Sobre o Projeto

Tem Quadra? é uma plataforma SaaS (Software as a Service) desenvolvida para resolver o problema de ociosidade e dificuldade de agendamento em complexos esportivos.

O sistema atende dois públicos:

Atletas: Que buscam quadras de Futebol, Vôlei, Beach Tennis, etc., e desejam agendar horário sem burocracia.

Donos de Quadras (Parceiros): Que precisam gerenciar a disponibilidade e divulgar seus espaços.

O foco principal do desenvolvimento foi a Responsividade Mobile (Mobile First), garantindo uma experiência nativa tanto em Android/iOS quanto em Desktops.

# Funcionalidades

- Para o Atleta (Frontend)

Busca Inteligente: Filtre quadras por modalidade (Futebol, Society, Beach Tennis, Vôlei, etc.) ou localização.

Interface Visual: Visualização de fotos das arenas em alta qualidade.

Agendamento Rápido: Seleção de data e horário em poucos cliques.

Responsividade: Layout adaptável para qualquer tamanho de tela.

- Para o Parceiro (Admin)

Painel de Controle: Cadastro de novas quadras.

Gestão de Preços: Definição de valor por hora.

Gerenciamento: Exclusão e edição de quadras cadastradas.

# Tecnologias Utilizadas

O projeto foi desenvolvido utilizando uma stack moderna e tipada:

Frontend:

React.js (Vite)

TypeScript

Tailwind CSS (Estilização "Utility-First")

Lucide React (Ícones)

Backend:

Node.js

Express

TypeScript

PostgreSQL (Banco de Dados Relacional)

Arquitetura:

MVC (Model-View-Controller) no Backend

Componentização no Frontend

# Layout e Design

O projeto utiliza Tailwind CSS para garantir consistência visual.

Cores primárias: Emerald-900 (Verde escuro institucional) e Yellow-400 (Amarelo vibrante para CTAs).

Hero Section: Imagem imersiva "Full Width" retangular para impacto visual imediato.
